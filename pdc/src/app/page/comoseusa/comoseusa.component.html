PASO 1:

Regístrate gratuitamente en PREDECONST, para posteriormente ingresar con tu nueva CUENTA DE USUARIO.

PASO 2:

Crea un PROYECTO y configura sus parámetros generales (I.V.A., I.T., tipos de cambio, etc.).

PASO 3:

Una vez creado el proyecto selecciónalo de tu lista personal, la aplicación te llevara a la gestión de módulos y crea uno.

PASO 4:

Agrega las ACTIVIDADES desde la base de datos de PREDECONST al módulo que creaste y comienza a armar el proyecto de la misma manera.

PASO 5:

Cuando estés listo, puedes usar las opciones de exportación ya sea para los precios unitarios y/o el presupuesto de insumos, la lista de módulos, etc.
